#### 3.阐述系统特性

①真实性：这些功能是人们尝使用到的，面对学习生活的方方面面需要记录的点滴，普通便签单调的格式无法满足多种需求。若要对症下药，多个软件又略有繁琐
②可用性：拥有良好的图形界面，清晰明了的功能模块，每个功能有独立格式
③价值所在：我们的想法是整合到一个软件，方便用户对多种功能一次到位的实现，可以避免当人们需要使用这些功能时需要下载多个软件
